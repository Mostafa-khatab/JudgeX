# ميزة ChatBot - واجهة بنمط LeetCode

## نظرة عامة
تضيف هذه الميزة واجهة لحل المشاكل البرمجية بنمط LeetCode مع مساعد ذكاء اصطناعي متكامل.

## المميزات

### 1. واجهة منقسمة
- **اللوحة اليسرى**: وصف المشكلة مع تبويبات:
  - الوصف: بيان المشكلة الكامل مع الأمثلة
  - التفاصيل: معلومات المشكلة (المعرف، النقاط، القيود، الإحصائيات)
  
- **اللوحة اليمنى**: محرر الأكواد مع:
  - دعم لغات متعددة (C, C++, Python)
  - تكامل مع Monaco Editor
  - تمييز الأكواد
  - أزرار التشغيل والإرسال

### 2. مساعد ChatBot الذكي
- **زر الدردشة العائم**: متاح دائماً في الزاوية السفلية اليمنى
- **يفهم السياق**: يعرف المشكلة الحالية والكود الخاص بك
- **إجابات مفيدة**: يمكنه تقديم:
  - تلميحات وإرشادات
  - مساعدة في تصحيح الأخطاء
  - اقتراحات للخوارزميات
  - تحليل التعقيد الزمني
  - شرح الأمثلة خطوة بخطوة

### 3. تجربة المستخدم
- **وضع ملء الشاشة**: للتركيز على محرر الأكواد فقط
- **دعم الوضع الداكن**: تكامل سلس مع السمات
- **تصميم متجاوب**: يعمل على جميع أحجام الشاشات
- **تحديثات فورية**: ردود وتعليقات فورية

## كيفية الاستخدام

### الوصول إلى صفحة الحل
1. انتقل إلى أي صفحة مشكلة
2. اضغط على زر "Solve" (يتطلب تسجيل الدخول)
3. سيتم توجيهك إلى `/problem/:id/solve`

### استخدام ChatBot
1. اضغط على زر الدردشة العائم في الزاوية السفلية اليمنى
2. اكتب سؤالك في حقل الإدخال
3. اضغط Enter أو اضغط على زر الإرسال
4. سيرد البوت بإرشادات مفيدة

### أمثلة على الأسئلة
- "Can you give me a hint?" (هل يمكنك إعطائي تلميح؟)
- "What's the time complexity?" (ما هو التعقيد الزمني؟)
- "Help me debug this code" (ساعدني في تصحيح هذا الكود)
- "Explain the algorithm" (اشرح الخوارزمية)
- "Walk me through an example" (اشرح لي مثال خطوة بخطوة)

## التنفيذ التقني

### مكونات Frontend
- **ProblemSolve.jsx**: المكون الرئيسي للصفحة
- **ChatBot.jsx**: واجهة الدردشة العائمة
- **chatbot.js**: خدمة API للرسائل

### Backend API
- **Endpoint**: `POST /chatbot/message`
- **المصادقة**: مطلوبة
- **البيانات المرسلة**:
  ```json
  {
    "message": "سؤال المستخدم",
    "problemId": "معرف-المشكلة",
    "code": "كود المستخدم",
    "language": "c++17",
    "history": []
  }
  ```

## التحسينات المستقبلية

### تكامل الذكاء الاصطناعي
حالياً، يستخدم ChatBot إجابات محددة مسبقاً. للتكامل مع ذكاء اصطناعي حقيقي:

1. **OpenAI**:
   - احصل على API Key من OpenAI
   - أضف `OPENAI_API_KEY` في ملف `.env`
   - عدّل `chatbotController.js` لاستخدام OpenAI API

2. **Anthropic Claude**:
   - احصل على API Key من Anthropic
   - أضف `ANTHROPIC_API_KEY` في ملف `.env`
   - عدّل `chatbotController.js` لاستخدام Claude API

## الاختبار

### خطوات الاختبار اليدوي
1. شغّل الخادم: `cd server && npm run dev`
2. شغّل العميل: `cd client && npm run dev`
3. سجل الدخول إلى حسابك
4. انتقل إلى أي مشكلة
5. اضغط على زر "Solve"
6. اختبر محرر الأكواد
7. اختبر ChatBot بطرح أسئلة
