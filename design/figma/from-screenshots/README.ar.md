حزمة استيراد لملفات الصور إلى فيجما

محتوى المجلد:
- ملفات SVG لكل لقطة شاشة (frame-1.svg, frame-2.svg, ...)
- سكربت PowerShell لفتح المجلد

الهدف:
تسهيل استيراد لقطات الشاشة إلى فيجما كـ Artboards/Frames بحيث يمكنك ترتيبها وإنشاء Components بسهولة.

كيفية الاستخدام (بالعربية):
1. افتح ملف فيجما جديد أو موجود.
2. افتح المجلد `design/figma/from-screenshots` في مستكشف الملفات.
3. اسحب كل ملفات PNG و SVG إلى مساحة العمل في فيجما (أو استخدم File → Import).
   - إذا قمت بسحب الـ PNG فقط: فيجما ستضع الصورة كـ Image، يمكنك تحويلها إلى Frame (Right-click → Frame selection).
   - إذا قمت بسحب الـ SVG: كل SVG يحتوي على Image التي تشير إلى ملف PNG في مجلد أعلى، لذلك اسحب PNG وSVG معًا للتأكد من أن الصورة تظهر صحيحة.
4. بعد الاستيراد، حدد كل عنصر وقم بإنشاء Components مشتركة (يمين → Create Component) للعناصر المتكررة (الهيدر، الشريط الجانبي، البطاقات).
5. أنشئ Color Styles و Text Styles يدوياً باستخدام القيم الموجودة في `design/figma/tokens.json`.

ملاحظة فنية:
- الملفات SVG تشير إلى الصور الأصلية عبر مسار نسبي. إذا لم تظهر الصور داخل فيجما بعد الاستيراد، اسحب ملف PNG إلى داخل الملف في فيجما وسيلتقط الصورة.
- إذا تريد، أستطيع دمج الصور مباشرة داخل SVG (base64) لكن ذلك سيزيد حجم الملفات.

أريدك أن تخبرني هل تفضل أن أدمج الصور داخل SVG مباشرة (base64) أم أن أعدّ ملف ZIP للتحميل؟
